# Saved CSS / Canvas Art — Endispute experiments

## ForceLattice.tsx  (from site-equilibrium)
Two opposing force-fields (the "two parties" of a dispute) of nodes drifting on a
flow field; nearby nodes connect with distance-faded lines. Same-side links are
the party hue (teal / amber), bridging links are indigo (the agreement forming).
- Cursor acts as a force that pulls nodes and brightens nearby threads (weaving).
- Scroll progress resolves the two separated clouds into one woven, balanced lattice.
- rAF only, no per-frame React state; honors prefers-reduced-motion + touch.
Palette: night #060a12, teal #4fd1c5, amber #f5a524, indigo #6366f1.
The client flagged this art as "really cool and awesome — save and explore later."
